# Lucera
Lucera - L’étiquette nutritionnelle de l’IA 

## Objectif
Lucera est un outil pour rendre visible la consommation énergétique des IA.  
Il permet aux startups et entreprises québécoises de comprendre et réduire l'impact de leurs modèles IA.

## Fonctionnalités MVP
- Simulation multi-thread des modèles IA
- Dashboard simple avec “étiquette énergétique” par requête
- Visualisation de consommation CPU/GPU estimée
- Transparence : révélation de mon propre usage AI à la fin du challenge

## Stack
- Backend : Python (simulation, utils)
- Frontend : HTML / CSS / JavaScript (dashboard interactif)
- Assets : images / reels / mockups

## Organisation
- `frontend/` → interface dashboard  
- `backend/` → simulation multi-thread et calcul consommation  
- `assets/` → sketches, Fujifilm photos, reels  
- `docs/` → whitepaper / résumé d’impact  
- `data/` → fichiers CSV de simulation
